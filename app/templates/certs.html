Pour utiliser le challenge DNS-01 (necessaire pour les certificats wildcard *.domaine), vous devez fournir vos identifiants API OVH.
Creez-les sur https://eu.api.ovh.com/createToken/ avec les droits :
GET /domain/zone/* — PUT /domain/zone/* — POST /domain/zone/* — DELETE /domain/zone/*
